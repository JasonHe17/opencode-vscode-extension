# OpenCode GUI Project

This repository contains the VSCode extension (opencode-gui) for OpenCode AI.

## Documentation

Please see the [extension/](./extension/) directory for the main project documentation:

- **Extension README**: [extension/README.md](./extension/README.md) - Extension-specific documentation
- **Development**: [AGENTS.md](./AGENTS.md) - Development guidelines
- **Changelog**: [extension/CHANGELOG.md](./extension/CHANGELOG.md) - Version history

## Quick Start

```bash
cd extension
bun install
bun run watch:esbuild
```

Then open `extension/` in VS Code and press `F5`.
